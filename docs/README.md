3. Read [`why/STRATEGIC_DIRECTION.md`](why/STRATEGIC_DIRECTION.md) for vision

**Implementing Web4?**
1. Review [`what/specifications/`](what/specifications/) for formal specs
2. Follow [`how/guides/`](how/guides/) for implementation patterns
3. Check [`reference/IMPLEMENTATION_STATUS.md`](reference/IMPLEMENTATION_STATUS.md) for current state

**Integrating with SAGE/ACT?**
1. See [`how/integration/`](how/integration/) for cross-system patterns
2. Check [`reference/ACT_QUICK_REFERENCE.md`](reference/ACT_QUICK_REFERENCE.md) for ACT details

**Security review?**
