Expert Review (RFC 8126)

## Error Classes

- 0x0000-0x00FF: Protocol errors
- 0x0100-0x01FF: Security errors  
- 0x0200-0x02FF: Trust errors
- 0x0300-0x03FF: Entity errors
- 0xF000-0xFFFE: Private use
- 0xFFFF: Reserved
